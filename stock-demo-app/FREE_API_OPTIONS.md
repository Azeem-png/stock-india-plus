# Free / Demo API Options

## For beta/prototype only
These can be explored for low-cost or free testing, but they are NOT enough to trust blindly for production stock intelligence.

### Market / stock data (beta experiments only)
- Alpha Vantage
- Twelve Data
- Financial Modeling Prep
- unofficial Yahoo wrappers

### News (beta experiments only)
- NewsAPI
- GNews
- Mediastack
- GDELT (harder cleanup, useful for experiments)

## Risks of free APIs
- low rate limits
- missing India coverage depth
- delayed data
- weak uptime guarantees
- legal/commercial restrictions
- no compliance support

## Best use
- demo
- private beta
- UI testing
- architecture validation

## Not best for
- financial production app with market-moving claims
