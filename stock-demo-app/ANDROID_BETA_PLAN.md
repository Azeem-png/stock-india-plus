# Android Beta Plan

## Goal
Turn Stock India Plus into an Android-only beta-ready project.

## What can be done right now
- prepare Android app architecture
- add screens and flows
- add repository/service layer
- add free/demo API placeholders
- keep TradingView deep links
- keep multilingual settings
- keep dark/light mode settings
- prepare AI ask interface with mock/demo backend contract

## What still blocks true production
- licensed Indian market data
- licensed news feeds
- final AI provider key + RAG backend
- Play Store release config/signing/privacy/compliance

## Recommendation
Use free/demo data in beta, then swap provider layer later.
